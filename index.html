Horizon is a realtime, open-source backend for JavaScript apps.

Horizon is a standalone server (and a collection of Node.js modules) with a simple JavaScript API to quickly build your app, then scale it to millions of users.

Built by the RethinkDB team and an open-source community, Horizon lets you build your apps with lightning speed.

Realtime fabric for your JavaScript app

Horizon's backend packs a realtime database, useful APIs and services, and makes it easy to use modern JavaScript tools like React, Angular, React Native, Cycle, and other popular frameworks.

Watch the screencast→

Scalable, realtime database

Horizon is built on RethinkDB, a massively scalable, open-source database that streams realtime updates, and scales linearly to millions of operations per second. Horizon stores JSON data, and pushes data and query results to clients in realtime.

Authentication

Authenticate users through Facebook, Twitter, Google, GitHub, and other services, and manage user accounts, sessions, and permissions through convenient APIs.

Modular and extensible

Horizon is distributed as a standalone server for browser-based JavaScript apps. When you're ready to write custom backend code, use the Horizon modules in Node.js to smoothly transition. RethinkDB is bundled with Horizon, so you can make direct database queries on the backend using the powerful query language ReQL, for flexible joins, aggregations, binary file support, and rich querying.

Built with love by the open source community

Developed by a core team of database experts and over 100 contributors from around the world, Horizon is shaped by developers like you participating in an open development process.
